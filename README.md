# Generate diff

Программа, которая выводит на экран разницу двух файлов форматов json, yaml


## Требования

- Python >=3.9
- Poetry


## Как установить
```bash
git clone https://github.com/Safintim/python-project-lvl2.git
cd python-project-lvl2
make install
```
